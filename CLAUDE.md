# バズ予報 — 프로젝트 컨텍스트

## 서비스 개요
- 서비스명: バズ予報 (버즈 요호)
- 한 줄 설명: 일본 유튜버 전용 AI 바이럴 콘텐츠 아이디어 발굴 도구
- 타겟 고객: 일본인 유튜버 (구독자 1천~10만명, 초보~중급)
- 핵심 가치: 일본어 완전 지원 + 경쟁사 대비 1/5 가격 + 심플한 UI

## 핵심 기능 (MVP)
1. 아웃라이어 탐색: 일본 유튜브에서 채널 평균 대비 10배 이상 조회된 영상 수집·분류
2. AI 아이디어 생성: 아웃라이어 영상 선택 시 일본어 제목·기획안 자동 생성
3. 니치 탐색: 경쟁이 낮고 조회수 높은 일본어 카테고리 추천
4. 쇼츠 전용 탐색: 일본 유튜브 쇼츠 바이럴 포맷 필터링

## 요금제
- 무료: 아웃라이어 탐색 하루 3회, AI 아이디어 월 10개
- 스탠다드: ¥500/월 — 무제한 탐색, AI 아이디어 월 100개, 쇼츠 탐색
- 프로: ¥1,200/월 — 전체 기능 무제한, 니치 탐색, 경쟁 채널 분석

## 경쟁사 차별화
- VidIQ: 일본어 부분 지원, 월 ¥1,100~ → 우리는 완전 지원, 더 저렴
- 1of10: 일본어 없음, 월 ¥4,500~, 무료 체험 없음 → 우리는 무료 플랜 제공
- 차별점 3가지: ① 일본어 완전 지원 ② 기능 3개로 심플 ③ 최저가

## 기술 스택 (목표)
- 프론트엔드(화면): Next.js + Tailwind CSS
- 백엔드(서버): Node.js + Express
- AI: Claude API (Anthropic)
- 데이터베이스(데이터 저장소): Supabase
- 배포(인터넷에 올리기): Vercel
- 결제: Stripe

## UI 원칙
- 언어: 일본어 완전 지원
- 톤: 친근하고 심플, 초보 유튜버도 바로 쓸 수 있게
- 색상: 기존 랜딩 페이지 다크 테마 유지 (배경 #0a0a0f, 보라 #7c6dfa)

## 개발 원칙
- 한 번에 하나의 기능만 완성하고 다음으로 넘어간다
- 각 기능은 실제로 작동하는 상태로 완성한다
- 코드는 한국어 주석으로 설명한다
